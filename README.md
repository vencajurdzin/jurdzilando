# Jurdzilando

## Project Description

Jurdzilando is a simple React + TypeScript application that displays products using TanStack Query for data fetching and MUI for UI components. The app includes a product list, product detail page, and basic error/loading handling.

## Features

- Responsive product grid listing
- Product detail view with image and description
- Data fetching with React Query (TanStack Query)
- Notistack for error handling
- UI components and styling using MUI with `sx` prop
- Routing with React Router

## Installation and start

1. Clone the repository:

```bash
git clone https://github.com/vencajurdzin/jurdzilando.git
cd jurdzilando
```

2. Install dependencies with pnpm and run project:

```bash
pnpm install
pnpm run dev
```

3. Open your browser at http://localhost:5173
